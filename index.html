Architectural Plan App

Read the tutorial.

Shapes:

Line Square Rectangle Polygon

Actions:

Move Line Move Square Move Rectangle Resize Line Resize Square Change Polygon Color

Tutorial

  1. Pembuatan Shapes
    1. Membuat Garis
      1. Pilih "Line" pada menu Shapes.
      2. Klik dan drag cursor pada canvas untuk mengatur ukuran garis yang dibuat, lalu lepas cursor untuk membuat garis.
    2. Membuat Persegi
      1. Pilih "Square" pada menu Shapes.
      2. Klik dan drag cursor pada canvas untuk mengatur ukuran persegi yang dibuat, lalu lepas cursor untuk membuat persegi.
    3. Membuat Persegi Panjang
      1. Pilih "Rectangle" pada menu Shapes.
      2. Klik dan drag cursor pada canvas untuk mengatur ukuran persegi panjang yang dibuat, lalu lepas cursor untuk membuat persegi panjang.
    4. Membuat Poligon
      1. Pilih "Polygon" pada menu Shapes.
      2. Klik pada canvas untuk membuat sudut pada poligon (minimal tiga sudut).
      3. Untuk menyelesaikan membuat sebuah polygon, pilih menu lain.
  2. Aksi yang Dapat Dilakukan pada Shapes
    1. Memindahkan Garis
      1. Pilih "Move Line" pada menu Action.
      2. Klik dan drag pada garis yang telah dibuat pada canvas, lalu lepas cursor untuk meletakkan garis.
    2. Memindahkan Persegi
      1. Pilih "Move Square" pada menu Action.
      2. Klik dan drag pada persegi yang telah dibuat pada canvas, lalu lepas cursor untuk meletakkan persegi.
    3. Memindahkan Persegi Panjang
      1. Pilih "Move Rectangle" pada menu Action.
      2. Klik dan drag pada persegi panjang yang telah dibuat pada canvas, lalu lepas cursor untuk meletakkan persegi panjang.
    4. Mengatur Ukuran Garis
      1. Pilih "Resize Line" pada menu Action.
      2. Klik dan drag pada salah satu ujung garis yang telah dibuat pada canvas, lalu lepas cursor saat sudah mendapatkan ukuran garis yang diinginkan.
    5. Mengatur Ukuran Persegi
      1. Pilih "Resize Square" pada menu Action.
      2. Klik dan drag pada salah satu sudut persegi yang telah dibuat pada canvas, lalu lepas cursor saat sudah mendapatkan ukuran persegi yang diinginkan.
    6. Mengganti Warna Poligon
      1. Pilih "Change Polygon Color" pada menu Action.
      2. Pilih warna pada Color Picker di sebelah kanan pilihan "Change Polygon Color"
      3. Klik pada salah satu sudut dari poligon yang ingin diubah warnanya.
  3. Download dan Upload File Gambar
    1. Download File Gambar
      1. Buat gambar pada canvas.
      2. Klik tombol "Download Architectural Plan" dan file gambar akan terunduh secara sendirinya dalam bentuk file .json.
    2. Upload File Gambar
      1. Klik tombol "Upload Architectural Plan".
      2. Pilih file architectural plan dalam bentuk .json yang telah diunduh sebelumnya.
      3. Klik "open" dan gambar yang diunggah langsung akan ditampilkan pada canvas
